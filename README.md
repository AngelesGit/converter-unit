# Converter Unit

COnvertet Unit is a simple unit converter app, that can be used to convert units of measurement between different units, built with [Scrimba](https://scrimba.com/) Frontend Developer Career Path.

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript
